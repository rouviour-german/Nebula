# Nebula Portfolio
> A stunning, immersive 3D developer portfolio showcasing the boundaries of modern web development.

![Nebula Portfolio Showcase](https://github.com/user-attachments/assets/d4148b6f-24da-412f-8cd8-c693d27f5bc7)

## 🚀 Overview
Nebula Portfolio is a high-end, immersive web experience designed to showcase developer skills and projects. Built with Next.js 14 and powered by Three.js, it takes users on a visual journey through space, combining storytelling with technical excellence.

## ✨ Features
- **3D Immersive Environment**: Interactive space scenes powered by Three.js and React Three Fiber.
- **Cinematic Animations**: Smooth, high-fidelity transitions using Framer Motion.
- **Performance First**: Optimized for Next.js 14 to ensure fast load times despite heavy assets.
- **Interactive UI**: Engaging user interface with modern icons and responsive layouts.
- **Email Integration**: Built-in contact functionality using EmailJS.

## 🛠️ Tech Stack
- **Framework**: Next.js 14
- **3D Engine**: Three.js, React Three Fiber, React Three Drei
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS
- **Icons**: Heroicons, React Icons
- **Language**: TypeScript

## 🌟 Live Site
View the immersive experience live: [Nebula Portfolio Demo](https://andrew-ichihara.vercel.app/)

## 📦 Installation
Clone the repository and install the dependencies:
```bash
npm install
```

## 💻 Usage
Run the development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
